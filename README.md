# flask-resume-analyzer
The following repository contains the code for an application named "Resume Analyzer" which is created using python and FastAPI

App Name: Resume Analyzer + Job Recommender Tech Stack:

Flask for backend

HTML/CSS + Bootstrap for frontend

SQLite / Supabase for database

spaCy + SentenceTransformer for NLP skill extraction

Job scraping API for LinkedIn/Naukri (using public job APIs or SerpApi)

Deployment: Render.com (free Flask app hosting)
